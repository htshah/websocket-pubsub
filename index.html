A PubSub based Websocket Library.
Light, Simple & Dependency-free.
Download 

      var ws = new WebsocketPubSub(
        'wss://echo.websocket.org'
      );
      
      var function1_key 
        = ws.subscribe('channel1', function(data){
        console.log('Yay got some data on channel1: ' + data);
      });

      ws.emit('channel1', 'Hey channel1!');

      ws.unsubscribe(function1_key);

Light

Very light-weight yet very powerful. Only about xkb in size.

Simple

A very simple API that anyone can use it easily in their projects.

Dependency-free

A completely dependency-free library making it smaller in size.

Options

reconnect

Default: true

Auto reconnects to Websocket server on connection lost.

attempts

Default: 5

Max. number of auto reconnect attempts to be made.

timeout

Default: 3000 (milliSeconds)

Auto close connection after a timeout if no connection can be made.

buffer

Default: true

If set, then all the messages will be buffered until the connection is re-established.
Methods

emit(ch, msg)

Returns: void

Emits msg on channel ch.

ch: [String] Required. Channel name.

msg: [String] Required. Message to be sent.

subscribe(ch, fn)

Returns: String

Subscribes to given channel and calls given function whenever there is new data on the channel.

ch: [String] Required. Channel name.

fn(data,event): [String] Required. Called when new data arrives for given channel.

unsubscribe(key)

Returns: void

Removes function from subscription list.

key: [String] Required. Value which is returned from subscribe function.
